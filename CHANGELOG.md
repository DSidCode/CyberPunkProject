# Changelog

Todas las cambios notables en este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), y este proyecto se adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2023-11-21

### Added
- Inicialización del repositorio en Git y configuración en GitHub.

## [1.0.0] - 2023-10-27

### Added
- Manifiesto poético como introducción artística al contenido.
- Contenido histórico sobre el cyberpunk, restaurado y colocado después del manifiesto.
- Tabla de contenido para navegar por las secciones históricas.

### Changed
- El título principal ahora es "// MANIFIESTO DEL NEÓN Y EL CROMO".
- El título de la tabla de contenido es "// Archivos Históricos" para mantener la temática.

## [0.5.0] - 2023-10-27

### Changed
- **BREAKING CHANGE:** Se reemplazó el contenido histórico por un manifiesto poético sobre la esencia del cyberpunk.
- Se eliminó la tabla de contenido y la estructura de secciones para adaptarse al nuevo formato lírico.

## [0.4.0] - 2023-10-27

### Changed
- Se mejoró la ergonomía de lectura del texto principal.
- Se aumentó el brillo del color de los párrafos para un mejor contraste.
- Se suavizó el color de los enlaces para reducir la fatiga visual.
- Se añadió sangría (`text-indent`) a los párrafos para facilitar el flujo de lectura.
- Se cambió el color del borde del contenedor a un cian oscuro menos agresivo.

## [0.3.0] - 2023-10-27

### Added
- Contenido informativo sobre la historia y evolución del término "cyberpunk".
- Tabla de contenido para navegar por las nuevas secciones.

### Changed
- Se reemplazó el texto de bienvenida genérico por el nuevo contenido histórico.

## [0.2.0] - 2023-10-27

### Added
- Efectos de animación "glitch" y "flicker" en el título principal usando CSS.

### Changed
- **Refactor:** Se movieron todos los estilos desde el archivo HTML a un archivo externo `style.css`.

## [0.1.0] - 2023-10-27

### Added
- Creación inicial del proyecto con un archivo `CyberPunk.html` que contiene HTML y CSS en línea.
- Estructura básica con un contenedor, título y párrafos de bienvenida.
